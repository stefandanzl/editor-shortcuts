# Editor Shortcuts

Edit your files like you're used to in VSCode

Below are the available Commands and personal recommendations for shortcuts

## Move Line Up/Down

Alt + Up/Down

## Duplicate Line

Ctrl + Shift + D

## Delete Entire Line

Ctrl + Shirt + Backspace

## Remove extra newlines after pasting

This is often times after pasting text from the web - especially from AI Chats - that there are empty lines between each line of text. Select the text and run command `Remove extra newlines after pasting` from Command Pallette
